# NightCompass Web 🌐

The official NightCompass marketing website, hosted at [nightcompass.net](https://nightcompass.net).

## Purpose
This repo contains the public-facing marketing site for NightCompass — designed to onboard Aurora chasers and night photographers onto the platform and communicate the product's value.

## Tech Stack
| Layer | Technology |
|---|---|
| Frontend | HTML, CSS, Vanilla JavaScript |
| Hosting | GitHub Pages |
| Domain | nightcompass.net |
| Fonts | Syne, DM Mono (Google Fonts) |

## Deployment
This site is deployed automatically via GitHub Pages from the `main` branch.

To configure:
1. Go to **Settings → Pages**
2. Set source to `main` branch
3. Set custom domain to `nightcompass.net`
4. Enable **Enforce HTTPS**
